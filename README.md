# Python-crawler-for-gamer-and-plurk
 從 Gamer(巴哈姆特) 和 Plurk(噗浪) 的帳號首頁取得資料，然後將之填進 Google Sheets 裡。  
 Get the data from account homepages of Gamer(巴哈姆特) and Plurk(噗浪) and then write it into Google Sheets.
